# Dating Sim Advanced

Liste d'idée : 

Le but du jeux serait de representrer la cité du pré gentil, avec potentiellement d'autre lieu iconique tel que le 22 rue chance milly, qui pourront ettre lié entre eux via un système de fast travel. On incarne un jeune lycéen de 17-18 ans qui doit passer le bac a la fin de l'année. Le joueur aura environ 300 jours pour choisir la fin qu'il souhaite pour son personnage 

Plusieurs fin possible selon le chemin emprunté :
- Boss du rap game
- Boss du rainté
- Marriage halal avec une meuf du coin
- Maitre kebabier
- Star du foot
- Purification par l'atome 
- La fac

Lieux de la cité :
- Batiment pour la vente de drogue
- Le QG (l'appartement)
- Le grec du coin (MAXIN CHIKEN, merci chef, maitre kebabier)
- Le studio d'enregistrement

- Cybercafé
- L'épicerie
